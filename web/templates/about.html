About the project

     This web application is designed to be a useful tool in the task of detecting events and anomalies in social media messages. It's development has been focused on analyzing Twitter data streams. It uses Python as back-end language, the data is stored in a NoSql database, MongoDb, and the graphical interface is made primarily with Jquery and Boostrap.

     After a user logs in or creates a new account, in the DashBoard page he can upload a file which contains the data he wishes to analyze, set his own custom configuration and go threw the steps of the application. The Results of this processing can be viewed in the Results page.
